myrts_flutter

MyRTS sdk for flutter.

Installation

Add this line into your pubspec.yaml

myrts_flutter:
    git:
        url: https://github.com/qcynaut/myrts_flutter.git

make sure you are added your ssh key into your ssh agent.

Usage

To connect to the MyRTS server, you can create MyRTS object like below example.

Future<void> connect() async {
    try {
        final url = "wss://socket.myrts.id"; // socket server of the MyRTS.
        final myrts = await MyRTS.create(url, 5);
        // the code above will try to connect to the MyRTS server at the given url, and we set the connection
        // timeout so it will not wait forever.
        // if the timeout has been reached, it will throw an error.
        setState({
            // do something after connection has been established.
        });
    } catch(err) {
        // do something with the error.
    }
}

Once we got the connection established, the next step we want to identified our device so we can begin using the MyRTS.

Future<void> auth() async {
    try {
        final id = "1234"; // the user id.
        final name = "John Doe"; // the user name.
        final result = await myrts.auth(id, name);
        // the code above will try to authenticate the user with the given id and name.
        // it will return AuthResponse object.
        if (result.success) {
            // do something after authentication has been established.
        } else {
            final message = result.message;
            // do something with the error.
        }
    } catch(err) {
        // do something with the error.
    }
}

After we got authenticated, we will be able to receive realtime data update via MyRTS event. This listener should be added before we authenticate the user.

final myrts = await MyRTS.create(url, 5);
myrts.onData((data) {
    // do something with the data.
});

Before we can start broadcasting, we need to create room for broadcast.

final res = await MyRTS.createRoom(name); // create a room.

After this room we can add or remove members to the room.

await res.addMember(["1234"]); // the addMember function receives list of user ids.
// it will return AddRoomMemberResponse object.

await res.removeMember(["1234"]); // the removeMember function receives list of user ids.
// it will return RemoveRoomMemberResponse object.

Now we can start the broadcast.

await res.startBroadcast();

And we can stop the broadcast.

await res.stopBroadcast();
